Description: This is one of my favorite projects because I made this site out of genuine interest of plants. I am an avid Plant Parent and wanted to combine my interest with coding! I enjoyed learning more about responsive design and how to use responsive design to promote efficiency.

Technologies: I built this project with HTML, CSS and Javascript.

Special Features: This site has hover effect and I used JavaScript animation for scroll effect using scroll reveal.
