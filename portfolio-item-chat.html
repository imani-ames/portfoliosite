ChatBox Messaging App

Facebook Messenger inspired ChatBot

Check It Out!

Description: This project is a chat bot inspired by the Facebook messenger application. This project allows users to chat with other users. This project also allows users to login with their facebook and google email accounts to access the chat page. The social logins are handled by Firebase social login for secure access.

Technologies: I built this project with HTML, CSS, Javascript, React.js, React Context API, Firebase API, ChatEngine API

Special Features: This project uses Firebase authentication for Facebook and Google login, as well as APIs. Another feature is that there is a login page that allows access to the chat room page.
